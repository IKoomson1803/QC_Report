Header

Alt + d then ENTER will take you to this page or clicking on HEADER in the menu bar on the right side. You can also click on the word HEADER in the Tree Frame (blue area).

The Header page displays fields for the following information which includes:

 
  • Client PO Number
  • QC Date
  • Client Name
  • Episode Number
  • Version
  • Standard
  • Subtitle
  • Language
  • Tape Desc.
  • Aspect Ratio
  • Title
  • Episode Name
  • Format
  • TRT
  • Tape Number
  • Status
  • Logo Details
  

A number of fields include pull down menus with predetermined choices. After entering the information please click the SAVE button to save your data.
