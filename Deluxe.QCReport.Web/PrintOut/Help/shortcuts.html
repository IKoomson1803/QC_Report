Keyboard Shortcuts

In the following keyboard shortcuts the + means to do the keystrokes as a combination.

Command

  

What it does.

  
alt+d then ENTER   Inside REVISIONS - takes user to the HEADER information page for that REV  
alt+y then ENTER   Inside REVISIONS - takes user to the HISTORY information page for that REV  
alt+t then ENTER   Inside REVISIONS - takes user to the TAPE LAYOUT information page for that REV  
alt+x then ENTER   Inside REVISIONS - takes user to the TEXT INFO information page for that REV  
alt+u then ENTER   Inside REVISIONS - takes user to the AUDIO & T.C. information page for that REV  
alt+i then ENTER   Inside REVISIONS - takes user to the INDIVIDUAL SPECS information page for that REV  
alt+o then ENTER   Inside REVISIONS - takes user to the OVERALL SPECS information page for that REV  
alt+b then ENTER   Inside REVISIONS - takes user to the BVHE SPECS information page for that REV  
alt+c then ENTER   Inside REVISIONS - takes user to the CTISS SPECS information page for that REV  
alt+l then ENTER   Inside REVISIONS - takes user to the LOG information page for that REV  
alt+f then ENTER   Inside REVISIONS - takes user to the FINAL information page for that REV
